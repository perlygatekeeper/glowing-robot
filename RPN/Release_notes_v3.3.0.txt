Release Notes
Version 3.3.0
=============

Overview
--------
Version 3.3.0 expands the statistics command set.

New Commands
------------

product
    Replaces the stack with the product of all values.

range
    Replaces the stack with maximum minus minimum.

median
    Replaces the stack with the median value.

variance / var
    Replaces the stack with the population variance.

stddev / stdev
    Replaces the stack with the population standard deviation.

Testing
-------
