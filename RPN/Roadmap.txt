RPN Calculator

## Current Version

3.0.0

## Project Goals

The goal of the RPN project is to create a powerful, programmable,
extensible Reverse Polish Notation calculator implemented in Perl.

The project began as a large monolithic script and has been refactored
into a fully object-oriented architecture consisting of:

rpn.pl
lib/RPN.pm
lib/RPN/Stack.pm
lib/RPN/Commands.pm

The emphasis is on:

* Maintainability
* Extensibility
* Testability
* Interactive use
* Scriptability
* Mathematical utility
