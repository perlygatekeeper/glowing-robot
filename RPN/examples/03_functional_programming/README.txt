03 Functional Programming
=========================

Recipes
-------

* Square Every Element: Demonstrates map.
* Keep Only Even Values: Demonstrates filter.
* Sum a Collection: Demonstrates reduce with addition.
* Product of a Collection: Demonstrates reduce with multiplication.
* Running Totals: Demonstrates cumulative thinking; may require future scan support.
* Composable Functions: Demonstrates combining reusable functions.
* Callbacks: Demonstrates executable values as deferred behavior.
