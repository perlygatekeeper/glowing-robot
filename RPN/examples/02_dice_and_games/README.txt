02 Dice And Games
=================

Recipes
-------

* Roll a d20: Demonstrates dieroll.
* Roll NdM Dice: Demonstrates repeated random rolls.
* Advantage / Disadvantage: Demonstrates max/min over dice rolls.
* 4d6 Drop Lowest: Demonstrates dice rolling, sorting, dropping, and reducing.
* Exploding Dice: Demonstrates conditional repetition.
* Percentile Dice: Demonstrates d100-style rolling.
* Dice Pool Successes: Demonstrates filtering rolls by target number.
* Random Ability Scores: Demonstrates repeating a recipe several times.
