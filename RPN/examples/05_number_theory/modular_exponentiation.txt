#------------------------------------------------------------
# Example: Modular Exponentiation
#
# Category:
#     05 Number Theory
#
# Purpose:
#     Demonstrates efficient repeated operations.
#
# Demonstrates:
#     exponentiate, modulo, composing arithmetic operations
#
# Notes:
#     This direct method is suitable for small exact integers. Large powers
#     need a future modular-power command to avoid constructing the full
#     intermediate value.
#
# Status:
#     Complete
#------------------------------------------------------------

# Compute (7 ** 4) modulo 13:
#
#     7 4
#     exponentiate
#     13
#     modulo
#
# Result:
#
#     9
#
# This illustrates composition clearly: exponentiate leaves 2401, then
# modulo reduces it to the remainder 9.
