# Useful local rearrangements:
#
#     exchange     swap the top two values
#     duplicate    copy the top value
#     over         copy the second value to the top
#     3 pullup     bring the fourth value to the top
#     3 pushdown   move the top value three places deeper
#
# Prefer these direct operations over moving values through a temporary
# variable when the values are already nearby.
