01 Arithmetic
=============

Recipes
-------

* Factorial with reduce: Demonstrates range, exchange, reduce, and executable values. Educational only; prefer the built-in factorial command for robust use.
* Greatest Common Divisor: Demonstrates iterative thinking and Euclid's algorithm.
* Least Common Multiple: Demonstrates composing arithmetic recipes.
* Integer Power: Demonstrates repeated multiplication.
* Running Product: Demonstrates accumulation by multiplication.
* Running Sum: Demonstrates accumulation by addition.
* Clamp a Value: Demonstrates bounding a value between a minimum and maximum.
* Minimum/Maximum of Two Values: Demonstrates stack manipulation and conditional execution.
