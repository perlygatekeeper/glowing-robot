RPN Cookbook Examples
=====================

These files are educational examples, not a standard library.

They are intentionally organized around familiar problems so users can learn
how RPN commands compose. Some examples may duplicate built-in commands, such
as factorial, because the goal is to demonstrate language techniques rather
than provide the most robust implementation.

Conventions
-----------

* Each chapter folder contains one example file per recipe idea.
* Each file starts with a standard header.
* Solution bodies are intentionally left as TODO stubs for later development.
* Examples may eventually be loaded with loadfuncs during experimentation, but
  they should not be treated as polished libraries.
