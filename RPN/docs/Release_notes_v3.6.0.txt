RPN Calculator
Release Notes
Version 3.6.0
=============

## Release Date

June 2026

## Overview

Version 3.6.0 introduces first-class vector support.

Vectors are now a native calculator data type alongside numbers
and strings. Vectors can be created, displayed, stored on the
stack, manipulated mathematically, and used by user-defined
functions.

This release lays the foundation for future matrix mathematics,
transformations, and graphics-oriented features.

## New Data Type

Vectors

Examples:

1 2 vec2

Result:

[1,2]

1 2 3 vec3

Result:

[1,2,3]

Vectors display naturally throughout the calculator:

[1,2,3]

## Vector Creation

New commands:

vector
vec2
vec3

Examples:

1 2 vec2

1 2 3 vec3

1 2 3 4 4 vector

## Vector Inspection

New commands:

dim

Examples:

[1,2,3] dim

Result:

3

## Vector Arithmetic

New commands:

vadd
vsub
vscale

Examples:

[1,2] [3,4] vadd

Result:

[4,6]

[5,7] [2,3] vsub

Result:

[3,4]

[3,4] 2 vscale

Result:

[6,8]

## Vector Products

New commands:

dot
cross

Examples:

[1,2,3] [4,5,6] dot

Result:

32

[1,0,0] [0,1,0] cross

Result:

[0,0,1]

## Vector Properties

New commands:

magnitude
normalize

Aliases:

mag
unit

Examples:

[3,4] magnitude

Result:

5

[3,4] normalize

Result:

[0.6,0.8]

## Display Improvements

Vectors are displayed using angle-bracket notation:

[x,y,z]

The calculator prompt, stack display, and diagnostic output
have been updated to display vectors cleanly.

## Error Handling Improvements

Numeric operations now preserve stack contents when invalid
operands are supplied.

Examples:

'hello sqrt

[1,2] add

The stack remains unchanged after the error.

This behavior improves calculator safety and prevents accidental
loss of data.
