RPN Calculator
Release Notes
Version 3.4.5
=============

Release Date
------------
June 2026


Overview
--------
Version 3.4.5 expands the calculator into a broader scientific,
engineering, financial, and number-theory toolset.

This release introduces financial functions, prime-number utilities,
scientific constant libraries, and improved help-system navigation.


New Financial Functions
-----------------------

Future Value

    fv

    pv rate nper fv

Present Value

    pv

    fv rate nper pv

Payment

    pmt

    pv rate nper pmt

Number of Periods

    nper

    pv rate pmt nper

Periodic Interest Rate

    rate

    pv fv nper rate


Examples

    1000 0.05 10 fv

    1000 1628.89462677744 10 rate

    1000 0.01 12 pmt

Help System Improvements
------------------------

New command:

    types

Displays all registered command categories.

Example:

    types

The following also works:

    help types
