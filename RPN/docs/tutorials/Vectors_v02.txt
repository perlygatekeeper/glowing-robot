# Vectors Tutorial

Vectors represent ordered collections of numeric values.

Unlike a single number, a vector contains multiple related values that are treated as a single mathematical object.

Examples:

[1,2]
[3,4]
[-5,10]
[1,2,3]

Vectors are commonly used in:

* Geometry
* Physics
* Engineering
* Computer graphics
* Navigation
* Machine learning

A vector can represent:

* A position
* A direction
* A velocity
* A force
* A color
* Any collection of related numeric values

---

## Creating Vectors

The simplest way to create a vector is using vector literal notation.

Example:

[1,2]

creates a two-dimensional vector.

Example:

[1,2,3]

creates a three-dimensional vector.

The dimension of a vector is the number of elements it contains.

Examples:

[5]

1-dimensional

[1,2]

2-dimensional

[1,2,3]

3-dimensional

---

## Displaying Vectors

A vector behaves like a single object on the stack.

Example:

RPN> [1,2]

Stack:

1: [1,2]

A vector occupies one stack entry regardless of its dimension.

---

## Vector Addition

Command:

vadd

Alias:

v+

Example:

RPN> [1,2]
RPN> [3,4]
RPN> v+

Result:

[4,6]

Each element is added independently:

[1+3, 2+4]

Vector addition requires vectors of the same dimension.

---

## Vector Subtraction

Command:

vsub

Alias:

v-

Example:

RPN> [5,7]
RPN> [2,3]
RPN> v-

Result:

[3,4]

Each component is subtracted independently.

---

## Scalar Multiplication

A vector can be multiplied by a scalar.

A scalar is simply an ordinary number.

Example:

RPN> [1,2]
RPN> 3
RPN> vscale

Result:

[3,6]

Every component is multiplied by the scalar.

Scalar multiplication changes the magnitude of a vector while preserving its direction.

---

## Dot Product

Command:

dot

Example:

RPN> [1,2]
RPN> [3,4]
RPN> dot

Result:

11

Calculation:

(1×3) + (2×4)

3 + 8

11

The dot product produces a scalar result rather than another vector.

The dot product is useful for:

* Measuring similarity
* Computing projections
* Determining angles
* Physics calculations

---

## Vector Magnitude

Command:

magnitude

Alias:

mag

Example:

RPN> [3,4]
RPN> mag

Result:

5

Calculation:

sqrt(3² + 4²)

sqrt(25)

5

The magnitude represents the vector's length.

For a vector:

[x,y]

the magnitude is:

sqrt(x² + y²)

---

## Unit Vectors

Command:

normalize

Example:

[3,4]
normalize

Result:

[0.6,0.8]

The resulting vector has magnitude 1.

Unit vectors preserve direction while removing scale.

---

## Distance Between Vectors

Command:

distance

Example:

[1,2]
[4,6]
distance

Result:

5

This is the Euclidean distance between the two points.

Distance is commonly used in:

* Navigation
* Geometry
* Computer graphics
* Machine learning

---

## Angles Between Vectors

Command:

angle

Example:

[1,0]
[0,1]
angle

Result:

90

assuming degree mode.

This indicates the vectors are perpendicular.

---

## Common Geometric Vectors

### X-Axis

[1,0]

Points directly right.

### Y-Axis

[0,1]

Points directly up.

### Diagonal

[1,1]

Points equally right and up.

### Three-Dimensional Example

[1,2,3]

Represents a point or direction in three-dimensional space.

---

## Vector Dimensions

Most vector operations require matching dimensions.

Valid:

RPN> [1,2]
RPN> [3,4]
RPN> v+

Invalid:

[1,2]
[3,4,5]
v+

because the dimensions differ.

---

## Vectors and Matrices

Vectors and matrices are closely related.

A matrix can transform a vector.

Example:

matrix
      ×
vector
      =
transformed vector

This is the foundation of:

* Computer graphics
* Rotation
* Reflection
* Scaling
* Coordinate transformations

The calculator supports matrix-vector multiplication through the matrix commands.

## Common Errors

### Dimension Mismatch

These vectors cannot be added:

[1,2]
[1,2,3]

because they have different dimensions.

### Zero Vector Normalization

The vector:

[0,0]

has magnitude zero.

It cannot be normalized because division by zero would occur.
