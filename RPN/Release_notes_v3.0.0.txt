RPN Calculator
Release Notes
Version 3.0.0
=============

## Release Date

June 2026

## Overview

Version 3.0.0 marks the completion of the major object-oriented
refactoring of the RPN Calculator.

The original single-file implementation has been replaced with a
modular architecture designed for long-term maintenance, testing,
and expansion.

## New Architecture

Project structure:

rpn.pl

lib/RPN.pm
lib/RPN/Stack.pm
lib/RPN/Commands.pm

Benefits:

* Reduced complexity
* Easier maintenance
* Improved testing
* Cleaner command implementation
* Better future extensibility
